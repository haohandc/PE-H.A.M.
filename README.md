## 项目说明
这是我个人做的一款PE文件分析&修改器，模仿了CFF Explorer，能实现基础的修改Characteristics和dllCharacteristics功能。
本项目源自大一的程序设计课程作业，是我第一次尝试编写正经的程序，里面不少内容是通过AI搜索，然后参考的。如有不足（当然有），还请见谅。
## About
This is an PE Header Analyzer & Modifier, which imitates CFF Explorer. But it is actually a project of one of my courses, and this is the first time I write a real program with GUI. So there maybe some(many?) mistakes.

# 特点
使用Qt6进行编写，拥有现代化的UI（相较于CFF Explorer）
缺点：只能运行在win10/win11系统
# Features
Qt6 is used for its GUI, which makes the UI more modern(in comparison to CFF Explorer)
